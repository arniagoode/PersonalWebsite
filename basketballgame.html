Tabletop Basketball Game

Role: Group project leader
Duration: Fall 2025

Project Overview

This tabletop basketball game features two lanes, each with its own launcher and an automatic scoring system that updates the scoreboard whenever a basket is made. The frame components were modeled in SolidWorks, laser cut from sheet material, and assembled into a rigid structure that supports the hoops, launchers, and electronic hardware. Infrared break-beam sensors mounted beneath each hoop detect successful shots and trigger the digital scoreboard, while beepers under each lane provide immediate audio feedback for every basket. A custom 3D-printed launcher, shaped with a precise curvature, functions as a spring-like mechanism to deliver consistent, repeatable shots toward the hoop.

This project was built for Art of Engineering, a required class for all Columbia Engineering students.

Skills Developed
SolidWorks, Laser Cutter, 3D Printing, Arduino IDE, C++, Soldering
