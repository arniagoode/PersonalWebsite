Addition & Subtraction Toy

Computer Graphics & Design Final Project
Spring 2026

Project Overview

This project is a fully mechanical toy designed to help children ages 5 and up learn basic addition and subtraction. The interface is divided into two lanes: the left lane represents the equation, and the right lane displays the solution. To use the toy, the child adjusts the handles on the left side to set their desired numbers. This action moves a duck forward by the corresponding number of spaces, visually representing the equation. The child then adjusts the handle on the right side to select their answer. If the answer is correct, the ducks align, providing immediate visual feedback.
