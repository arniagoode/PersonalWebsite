SenseWear

Role: Team member
Duration: 48 hours - Hardware Hackathon

Project Overview

SenseWear is a wearable navigation assistance device designed to help users detect nearby obstacles through haptic and audio feedback. The system features two ultrasonic sensors mounted on servo motors, one positioned on each shoulder, allowing each unit to scan a 180-degree field of view. Together, the sensors provide a wide coverage area, detecting objects within a 2-foot range. As each sensor scans its field, vibration motors near the detected object are activated at every 90-degree interval, giving the user intuitive directional cues for safer navigation.

This project won best beginner hack at the MakeCU 2025 Hardware Hackathon. Check out the full project description on DevPost: https://devpost.com/software/sensewear
